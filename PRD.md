# Admin System — Claude Code 구현 지시문

아래 요구사항에 따라 Web Application Template을 구현해줘.

## 구현 모듈 상세 (총 12개)

모든 모듈은 Mock 데이터 기반으로 UI까지 완성해줘.

---

### 1. Auth (인증)

- 로그인 페이지 (`/login`) — 이메일 + 비밀번호
- NextAuth.js Credentials Provider
- RBAC 4단계: `SUPER_ADMIN > STAFF > CLUB_LEADER > MEMBER`
- 미들웨어로 권한별 라우트 보호
- 비밀번호 초기화 UI (이메일 발송 폼)

---

### 2. Dashboard (`/dashboard`)

- KPI 카드 5종
  - 전체 회원 수
  - 현재 운영 기수
  - 이번달 출석률
  - 납부율
  - 임기 만료 임박 임원 수 (30일 이내)
- 이번달 일정 리스트 (교육 세션 + 행사 통합)
- 미납 현황 알림 배너
- 최근 공지사항 3건
- 다가오는 행사 카드

---

### 3. Members (회원 관리) — `/members`

- 목록: 테이블 뷰 ↔ 명함첩 카드 뷰 전환 버튼
- 검색: 이름 / 회사명
- 필터: 기수 / 상태(재학/수료/휴학) / 동호회
- 회원 상세 슬라이드 패널 (우측에서 열림)
  - 탭 구성: 기본정보 / 동호회 가입현황 / 납부현황 / 출석현황
- 회원 등록 / 수정 모달 폼
  - 필드: 이름, 사진, 소속회사, 직위, 이메일, 전화번호, 기수, 메모
- 엑셀 일괄 등록 / 전체 내보내기 버튼 UI (xlsx 라이브러리)
- 비활성화(탈퇴) 처리 — 물리 삭제 아닌 상태 변경

---

### 4. Cohorts (기수 관리) — `/cohorts`

- 기수 목록 카드 그리드
  - 상태 배지: 모집중(파랑) / 운영중(초록) / 수료(회색)
  - 카드 정보: 기수명, 운영기간, 정원/현재인원, 수료기준 출석률
- 기수 개설 / 수정 모달 폼
  - 필드: 기수명, 운영기간(시작~종료), 정원, 수료 기준 출석률(%), 담당자, 메모
- 기수 상세 페이지 (`/cohorts/[id]`)
  - 탭 구성
    - 소속 회원 목록 (회원 배정/해제)
    - 커리큘럼 (연결된 세션 목록)
    - 대표단 (기수장 / 부기수장 / 총무 역할 지정)

---

### 5. Sessions (교육 일정) — `/sessions`

- 상단 뷰 전환: 캘린더 뷰 / 리스트 뷰
- 캘린더 뷰: react-big-calendar, 월별/주별 전환, 교육 세션(파랑) + 행사(주황) 통합 표시
- 리스트 뷰: 테이블 (일시, 장소, 강사, 주제, 기수, 상태)
- 세션 등록 / 수정 모달
  - 필드: 일시, 종료시간, 장소, 강사 선택, 주제, 연결 기수, 상태(예정/완료/취소), 메모
- 강사 관리 서브페이지 (`/sessions/instructors`)
  - 외부 강사 목록, 등록/수정 폼 (이름, 소속, 전문분야, 연락처, 강의 이력)

---

### 6. Attendance (출결 관리) — `/attendance`

- 세션 선택 드롭다운 → 해당 기수 회원 목록 자동 로드
- 출석 체크 UI: 회원별 버튼 토글 (출석/지각/결석/공결 — 색상 구분)
- 일괄 처리 버튼: 전체 출석 / 전체 결석
- 회원별 누적 출석률 테이블
  - 컬럼: 이름 / 총 세션 수 / 출석 / 지각 / 결석 / 공결 / 출석률 / 수료가능여부
  - 수료 기준 미달 시 빨간 배지
- 기수별 출석 현황 요약 바 차트 (Recharts)

---

### 7. Board (게시판) — `/board`

- 상단 탭: 공지사항 / 강의자료 / 갤러리

**공지사항 탭**
- 목록 테이블 (제목, 작성자, 작성일, 조회수, 중요공지 여부)
- 중요 공지 상단 고정 + 빨간 배지
- 상세 페이지 (본문, 첨부파일)
- 작성 / 수정 폼 (제목, 본문, 중요공지 체크박스, 첨부파일)

**강의자료 탭**
- 파일 목록 (파일명, 관련 세션, 업로드일, 용량, 다운로드 버튼)
- 파일 업로드 드래그앤드롭 UI

**갤러리 탭**
- 이미지 카드 그리드
- 클릭 시 라이트박스
- 앨범 단위 분류 (행사명 / 기수명)

---

### 8. Finance (회비·납부 관리) — `/finance`

- 상단 탭: 교육 회비 / 행사 참가비

**교육 회비 탭**
- 기수별 회비 금액 및 납부 기한 설정 카드
- 회원별 납부 현황 테이블
  - 컬럼: 이름 / 기수 / 납부금액 / 납부일 / 상태(납부/미납/분할)
  - 상태별 색상 배지
- 납부 등록 모달 (회원 검색, 금액, 납부일, 납부방법, 비고)
- 미납자 리스트 탭 + 알림 발송 버튼 UI (SMS/이메일)
- 기수별 납부율 요약 카드 (납부인원/전체인원, 진행률 바)

**행사 참가비 탭**
- 행사별 참가비 현황
- 회원별 납부/미납 상태

---

### 9. Clubs (동호회 관리) — `/clubs`

- 동호회 목록 카드 그리드
  - 카드 정보: 동호회명, 아이콘, 소개, 회원 수, 동호회장
- 동호회 개설 / 수정 / 해산 폼
  - 필드: 이름, 아이콘, 소개, 동호회장 지정, 가입비, 정기모임 일정
- 동호회 상세 페이지 (`/clubs/[id]`)
  - 탭 구성
    - 회원 목록 (가입/탈퇴 처리)
    - 활동 일지 (일시, 제목, 내용, 사진 등록)
    - 회비 관리 (동호회 자체 회비 납부 현황)
- 예시 동호회 4개: 산악회 / 골프회 / 트레킹회 / 독서회

---

### 10. Events (행사 관리) — `/events`

- 행사 목록 카드 그리드 + 리스트 뷰 전환
- 행사 유형 필터 탭: 전체 / 워크숍 / 골프대회 / 체육대회 / 해외연수 / 송년회 / 기타
- 행사 등록 / 수정 폼
  - 필드: 행사명, 유형, 일시, 장소, 대상(전체/기수선택/동호회선택), 정원, 참가비, 신청마감일, 설명
- 행사 상세 페이지 (`/events/[id]`)
  - 탭 구성
    - 참가 신청 명단 (신청자 목록, 상태: 신청/확정/취소)
    - 출석 체크 (행사 당일 참석 확인)
    - 결과 보고 (사진, 후기 등록)
- Sessions 캘린더 및 Dashboard 일정 위젯에 행사 통합 표시 (주황색 구분)

---

### 11. Organization (조직 관리) — `/organization`

**조직도 홈 (`/organization`)**
- @xyflow/react 로 시각적 트리 조직도 구현
- 계층 구조: 명예회장 → 회장 → 수석부회장 → 부회장 → (이사 / 감사) → 집행부
- 노드 클릭 시 해당 임원 상세 정보 툴팁 또는 슬라이드 패널 표시
- 인쇄 버튼 (window.print + print CSS로 조직도만 출력)

**임원진 (`/organization/executives`)**
- 상단 탭: 현직 임원 / 역대 임원
- 현직 임원 테이블
  - 컬럼: 직책 / 성명 / 소속회사 / 임기시작 / 임기종료 / 상태 / 액션
  - 직책 순서 고정 정렬: 명예회장 → 회장 → 수석부회장 → 부회장 → 이사 → 감사
  - 임기 만료 30일 이내 행 노란 하이라이트 + 경고 아이콘
- 임원 등록 / 수정 모달
  - 필드: 회원 검색 연결, 직책 선택, 임기시작일, 임기종료일, 비고
- 역대 임원 탭: 전체 임원 이력 테이블 (기간 필터 포함)

**운영위원회 (`/organization/committee`)**
- 분과 탭: 기획분과 / 교육분과 / 친목분과 / 홍보분과
- 각 분과: 분과장 + 위원 목록 (회원 연결, 역할 입력)
- 위원 추가 / 수정 / 삭제
- 회의록 섹션
  - 회의록 목록 (일시, 안건, 참석자 수)
  - 회의록 등록 폼 (일시, 참석자 체크, 안건, 결과, 첨부파일)

**집행부 (`/organization/executive-board`)**
- 현재 기수 집행부 기본 표시, 상단에서 기수 선택 전환 가능
- 역할별 카드 그리드: 총무 / 재무 / 교육 / 홍보 / 섭외 / 행사 / 복지
  - 각 카드: 역할명, 담당자명, 연락처, 담당 업무 요약
- 담당자 지정 (회원 검색 연결) / 수정 모달
- 업무 인수인계 메모 텍스트 에디터 (각 역할별)
- 기수별 집행부 구성 이력 조회

---

### 12. Settings (시스템 설정) — `/settings`

- 탭 구성: 단체 기본정보 / 계정 관리 / 알림 템플릿

**단체 기본정보 탭**
- 로고 이미지 업로드
- 단체명, 설립일, 대표 연락처, 주소, 이메일, 홈페이지

**계정 관리 탭**
- 시스템 사용자 목록 (사무국 계정 등)
- 컬럼: 이름 / 이메일 / 권한(RBAC) / 상태 / 최근로그인
- 계정 등록 / 권한 변경 / 비활성화

**알림 템플릿 탭**
- SMS 템플릿 편집 (납부 안내, 일정 안내, 출석 독려)
- 이메일 템플릿 편집
- 변수 치환 안내 표시 (예: `{{이름}}`, `{{기수}}`, `{{금액}}`)

## 공통 구현 요구사항

### 레이아웃
- 좌측 사이드바 네비게이션 + 상단 헤더 고정
- 사이드바 메뉴 구성 (한국어, 아이콘 포함)
  - 대시보드
  - 회원 관리
  - 기수 관리
  - 교육 일정
  - 출결 관리
  - 게시판
  - 회비 관리
  - 동호회 관리
  - 행사 관리
  - 조직 관리 (서브메뉴: 조직도 / 임원진 / 운영위원회 / 집행부)
  - 시스템 설정
- 모바일: 사이드바 → 햄버거 버튼 드로어 전환
- 반응형 완전 지원 (모바일 / 태블릿 / 데스크탑)
- 다크모드 지원 (Tailwind `dark:` 클래스 + next-themes)

### 한국화
- 모든 UI 텍스트 한국어
- 날짜 형식: `YYYY년 MM월 DD일` 또는 `YYYY.MM.DD`
- 전화번호 형식: `010-0000-0000`
- 금액 형식: `1,000,000원`

### 스키마
아래 모델을 포함한 완전한 schema를 작성해줘.

```
User, Member, Cohort, CohortMember,
Session (교육세션), Instructor, Attendance,
Notice, BoardFile, GalleryImage,
Payment, CohortFee,
Club, ClubMember, ClubActivity, ClubFee,
Event, EventParticipant,
Executive, CommitteeDivision, CommitteeMember,
MeetingMinutes, ExecutiveBoardMember
```

---

위 내용을 기반으로 프로젝트를 생성하고,  
**각 페이지가 Mock 데이터로 실제 동작하는 UI를 갖추도록** 구현해줘.
